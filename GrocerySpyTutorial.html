Welcome To The Grocery Spy Tutorial!


First, Click "Start Here" to begin






This is the home screen for grocery Spy. Click "Browse All" to view the recipes currently in the database.




Here we can see our recipes. We'll come back to this page in a minute. Click the back button in the top right, and then click "Add Recipe"
to add a new recipe to the databse.




Here you can add recipes to the database. Lets start by filling in the Ingredient List.




Grocery Spy accepts copy and paste from the clip board. To add a recipe copy and paste each element of the recipe into the specified field.
We will start with the ingredients.




Because humans can write the same thing in many different ways, not all ingredient lists will be understood by Grocery Spy. For example,
ingredients written: # (amount) will not work. If we rewrite these ingredients
to take out the parenthesis, Grocery Spy will be able to understand the ingredient list. The spacing and other ingredients will be
handled just fine.




After copying our cleaned up ingredient list, simply press control + v to paste it into the Grocery Spy window. Then click Add Entry




If all goes well, the field we just filled in will now have a check mark on it. Complete the other fields in the same way, just by copy
and pasting. Don't worry about spacing or line breaks, Grocery Spy will automatically format your input. You can leave any field except
the ingredient list blank, by simply hitting "add entry" without copy and pasting.





Once all fields have been entered, the "submit recipe" button will appear. Click here to add your recipe to the database.




Return to the "browse all" page, and we can see our new recipe has been added.




Click the arrow button next to any recipe to view more information about that recipe.




More information about our new Cuban Ropa Vieja recipe.




Its difficult to view our recipe inside the Grocery Spy window. We can scroll up or down using the arrow keys, but it would be easier
to have this all in its own window. Select the pen and paper icon to create .txt file on the desktop.




A view of the desktop with the new recipe in text file form.




And this is what the text file looks like.




Now for my original reason to create this program. I always find that I go to the grocery store, buy everything I think I need
then when I get home I realize I forgot one thing, so I have to go back. Or I make 3 trips during the week, each time I only buy
what I need for the one recipe I'm cooking that day. With Grocery Spy, this hassle can be avoided.

Let's say I want to cook Cuban Ropa Vieja, Homemade Vegetable Soup and Slow Cooker Texas Pulled Pork this week. I can select the cart icon
(which will switch to the check mark to show that the recipe has been added), as shown in the above picture.

Return to the main menu using the back button in the top right, and then select "view cart." We can see all the ingredients we need to
buy the recipes we have selected! No more forgetting one random thing, we now have it all on one list. Grocery Spy will add up matching
ingredients, so if two recipes need a clove of garlic, it will combine them into 2 cloves of garlic as seen here. Also, if at any time
you decide you don't want to cook a recipe after all, you can navigate back to the "browse all" screen and unselect it by clicking the
cart button again. This will remove all ingredients for that recipe from your grocery list.
When you are satisfied with your grocery list, click the pad and pen.




Back to the desktop, Grocery Spy has put our new grocery list into a .txt file for easy viewing and printing.




And here is the text file




So lets say you our Cuban Ropa Vieja didn't taste so good. We can delete the recipe from our database by selecting the "delete recipe"
button. Clicking the red X will make it turn blue, which deletes the recipe. Once we click the back button, all selected recipes will
be deleted.




And here is our new recipe list, with the deleted recipes removed
